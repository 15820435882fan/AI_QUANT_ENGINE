# 📚 AI量化交易系统使用指南

## 快速开始
1. 启动系统: `python start_production.py`
2. 查看状态: `python monitoring_system.py`  
3. 优化参数: `python signal_optimizer.py`
4. 系统健康检查 python system_health_check_final.py
5. 运行最新优化回测 python high_frequency_backtest.py
6.  验证优化效果  python high_frequency_backtest.py

## 核心功能
- 自动策略优化
- 多策略组合管理
- 实时交易决策
- 系统健康监控

## 配置说明
- 修改 `config/production_config.json`
- 调整风险参数和交易阈值
